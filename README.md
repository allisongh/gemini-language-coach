# language-coach
Create your own AI language coach in 5 minutes using the Gemini app. You can use these instructions
to create [custom Gems](https://support.google.com/gemini/answer/15236321) that help you learn
French vocabulary, practice conversations, and get coached on your writing.

![AI language coach](https://miro.medium.com/v2/resize:fit:1400/format:webp/1*u1OrNAWWISfkfjCkoa-JoQ.png)

## What you need

To use the Gemini app, you must [sign in to a Google Account](https://support.google.com/gemini/answer/13278668).

## Create a Gem for language learning

To add this functionality in the Gemini app:

1. Go to gemini.google.com
1. From the menu, select **Settings & help** > **Instructions for Gemini**.
1. Click **Add **and enter one of the following instructions:
   * [Translator](https://github.com/allisongh/language-coach/blob/main/translator.md): Supercharge and localize language translation with a fancy custom prompt to Gemini.
   * [Restaurant Simulator](https://github.com/allisongh/language-coach/blob/main/restaurant-simulator.md): Practice ordering at any restaurant in France.
   * [Essay coach](https://github.com/allisongh/language-coach/blob/main/essay-coach.md): Refine your writing style through customized topics, vocabulary, and context.
